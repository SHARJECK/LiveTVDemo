# 电视语音换台源码对接

#### 介绍
电视语音换台源码参考，动态对接夏杰语音，直接支持语音换台。

#### 软件架构
软件架构说明


#### 使用说明

1.  下载源码编译，也可以合入到自己工程里面。
2.  从当贝市场下载安装夏杰语音【永久免费】。
3.  运行LiveTV软件，会动态注册到夏杰语音，即可实现语音换台。
【支持手机控制、微信小程序控制、智能音箱控制等等。】
